# SQL Database Practice Repository

## 📌 Overview
This repository contains SQL scripts demonstrating fundamental and advanced database concepts. It is designed to showcase practical knowledge of SQL for real-world applications.

## 🛠️ Topics Covered
- Data Definition Language (DDL)
- Data Manipulation Language (DML)
- Joins (Inner, Left, Right, Full)
- Subqueries (Correlated & Non-Correlated)
- Stored Procedures

## 📂 Folder Structure
- **DDL/** – Table creation and schema design
- **DML/** – Insert, update, delete operations
- **Joins/** – Different types of joins with examples
- **Subqueries/** – Nested queries and use cases
- **Stored_Procedures/** – Procedural SQL examples
- **Sample_Database/** – Base schema used in scripts

## 🚀 How to Use
1. Clone the repository
2. Run `schema.sql` to create the database
3. Execute scripts folder-wise to understand concepts

## 💡 Purpose
This project is created as part of preparation for technical interviews and to demonstrate SQL proficiency for roles such as Software Engineer and Database Developer.
